# AI Agent 设计模式

## ReAct 模式

ReAct (Reasoning + Acting) 是一种将推理和行动结合的模式：

1. **思考 (Thought)**：分析当前情况
2. **行动 (Action)**：执行工具调用
3. **观察 (Observation)**：获取执行结果
4. **循环**：重复直到完成任务

## Plan-and-Execute 模式

先规划，后执行：

1. **规划阶段**：将任务分解为子任务
2. **执行阶段**：按顺序完成子任务
3. **验证阶段**：检查结果是否符合预期

## Multi-Agent 协作

多个代理协同工作：

- **主管代理**：分配任务
- **专业代理**：执行特定任务
- **审查代理**：检查结果

## 工具使用模式

### 工具选择策略
- 基于描述匹配
- 基于历史成功率
- 基于任务类型

### 错误处理
- 工具调用失败重试
- 备选工具切换
- 人工介入请求
