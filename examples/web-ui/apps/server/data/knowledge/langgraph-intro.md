# LangGraph 简介

LangGraph 是一个用于构建复杂 AI 代理工作流的 Python 库。

## 核心概念

### 状态管理
LangGraph 使用状态机模式管理代理状态，每个节点可以读取和修改共享状态。

### 图结构
工作流被建模为有向图，其中：
- **节点 (Nodes)**：代表代理、工具或函数
- **边 (Edges)**：代表状态转换

### 循环支持
与其他框架不同，LangGraph 原生支持循环，适合构建多步骤推理流程。

## 主要特性

1. **持久化**：内置状态持久化，支持断点续传
2. **人机协同**：支持 Human-in-the-loop 模式
3. **流式输出**：实时流式返回中间结果
4. **可视化**：内置图结构可视化工具

## 使用场景

- 多步骤研究代理
- 代码生成与审查
- 对话系统
- 自动化工作流
